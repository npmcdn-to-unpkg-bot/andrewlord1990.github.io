RDA (Running Digital Assistant)

RDA is a running assistant app for Android.

An app written as a group for a university project. RDA monitors your runs, detecting if you are running a previous route or a new one. It features text-to-speech feedback on your progress as well as music playback controlled with motion controls. Music genres can be linked to different running speeds, for selecting the next track.

  • Language: Java
  • Platform: Android
  • Availability: University project
