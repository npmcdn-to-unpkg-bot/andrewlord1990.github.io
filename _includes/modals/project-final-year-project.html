Thesis Project

A Reinforcement Learning framework for the game, Robocode

Robocode is an open source Java game, that allows programmed robots to fight each other in battles. My thesis project consisted of research into using Reinforcement Learning algorithms to learn a robot behaviour for the game Robocode. Development consisted of a framework to allow people to play robots that have learned their behaviour in the game. The Reinforcement Learning algorithm managed to learn a behaviour that could beat many advanced hand-coded robots.
