Ray Tracer

A ray tracing application, written in C++ at university

A programming project in computer graphics. The ray tracer is capable of producing a rendered image of the scene. Many features were implemented including various 3D shapes, lighting, shadows, reflections, Lambert and Phong materials, bounding volumes and flat triangle meshes.

  • Language: C++
  • Platform: Linux, Windows, Mac
  • Availability: University project
