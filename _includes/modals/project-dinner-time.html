Dinner Time

Dinner Time is a meal-timing assistant app for Android.

  • Uses the latest Android features: Navigation Drawer, Fragments, Card Views and Material Design.
  • The concept is for you to be able to setup multiple timers for each component of a meal and for the app to tell you when each part needs to start cooking. The aim is for it to take the complexity out of timing complex meals.
  • Language: Java, Android
  • Platform: Android
  • Availability: In Development
