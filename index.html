Skills

Java and Android

I develop Android apps from scratch and have been programming in Java since the start of university. Currently, I am an Android Software Engineer at Ocado Technology.

C# and Unity3D

I developed an Xbox 360 game in C#, as part of a team at university. More recently, I created a mobile game in Unity3D (using C#).

HTML and CSS

I design responsive websites using HTML, CSS and JavaScript. I designed and developed this website, which scales appropriately for different screen sizes.

Python and Shell

I have a small amount of experience in Python and Shell scripting. Most recently, I wrote a shell script to resize Android drawables and am planning to rewrite it in Python.

Git source control

I use Git to manage all of my programming projects. Some of my projects are hosted on Github, on which I also contribute to open source projects.

Android Studio

I use Android Studio for all of my Android development, using the Gradle build system. This has given me some experience in writing custom Gradle script behaviour.
